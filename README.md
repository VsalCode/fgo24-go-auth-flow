## How to Run This Project

1. Clone the repository:
   ```bash
   git clone https://github.com/VsalCode/fgo24-go-auth-flow.git
   ```

2. Navigate to the project folder:
   ```bash
   cd fgo24-go-auth-flow
   ```

3. run the program:
   ```bash
   go run main.go
   ```
